# Changelog - 2026-04-16

## Jobs Page & Pinpoint API Integration

### Overview

Completed the portfolio job aggregator by adding **Pinpoint** as a third job board API provider. All three portfolio companies — Anduril (Greenhouse), Hadrian (Ashby), and Impulse Space (Pinpoint) — are now live with a combined **194 job listings**.

---

## What Changed

### Pinpoint API Integration

- Added Pinpoint job board API support (free JSON endpoint at `postings.json`)
- Impulse Space now pulling 141 jobs with structured salary data
- Auto-detection of Pinpoint provider in the careers URL validator

### HTML Snippet Handling

- Job description snippets now preserve HTML formatting
- Universal snippet sanitization at the dispatcher level: decodes HTML entities, strips boilerplate, works across all providers
- Job cards render snippets with `set:html` for proper formatting
