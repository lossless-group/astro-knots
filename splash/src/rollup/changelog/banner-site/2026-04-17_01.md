# Changelog - 2026-04-17

## New Flare Components & Simplified Homepage

### Overview

Two new flag-themed Three.js flare components join the design system, the homepage is stripped to a radically simple hero, and the header is trimmed to just two links per client request.

---

## New Flare: FlagCloth

Particle-based cloth simulation — ~4,000 round dots form a grand banner billowing in wind.

- Left edge pinned to flagpole, trailing edge free
- Three-harmonic wind for organic chaotic motion
- Z-depth billowing, gravity sag, edge ripple
- Mouse parallax rotates the flag in 3D
- Props: `bgColor`, `dotColor`, `height`, `parallax`, `density`, `windStrength`

## New Flare: FlagPixel

Dense mosaic flag — ~8,400 square pixels form a solid-looking banner with clean, linear wave motion.

- InstancedMesh rendering (single draw call) for performance
- Linear freedom ramp from pole (more structured than FlagCloth)
- Each pixel tilts to match wave surface normal
- Per-instance color variation for subtle texture
- Thin grid lines between pixels (~18% gap)
- Props: `bgColor`, `pixelColor`, `height`, `parallax`, `density`, `windStrength`

## Simplified Homepage

- Replaced BannerFlag with FlagPixel as hero flare
- Removed gradient background (client preference)
- Removed hero text/CTA section for radically simple look
- Extracted hero content to `HeroMessage.astro` component for future use

## Trimmed Header

- Reduced navigation to two links: **Jobs** and **Investor Login** (external to arkpes.com)
- Removed mobile hamburger menu (two links fit all screen sizes)
- Removed dropdown menus
- Archived previous header as `HeaderArchive1.astro`

## Layout Adjustments

- Removed forced `min-height: 100vh` and flex layout from body/main
- Removed margin-top from footer for tighter spacing
- Flare gallery updated with both new components
