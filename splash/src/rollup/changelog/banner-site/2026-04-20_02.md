# Changelog - 2026-04-20 (02)

## Wobble Rings Flare Component

### Overview

A new creative direction for the Banner VC homepage — interactive concentric rings radiating from an off-center origin, with organic wobble and mouse-driven interactions. Built with HTML5 Canvas 2D and a self-contained simplex noise implementation (zero external dependencies).

---

## New Flare: WobbleRings

Concentric rings with a topographic-map / fingerprint aesthetic.

- **Origin point** at configurable rule-of-thirds position (default 33%, 33%)
- **Simplex noise wobble** — each ring's radius is perturbed per-angle with unique noise seeds, animated over time
- **Hover repulsion** — cursor between rings pushes them apart with gaussian falloff and directional cosine weighting
- **Click ripple** — shockwave propagates outward at 400px/s with decaying sine displacement over 2 seconds
- **Props**: `bgColor`, `ringColor`, `ringOpacity`, `ringGap`, `ringStrokeWidth`, `originX`, `originY`, `wobbleAmount`, `wobbleSpeed`, `hoverForce`, `clickRipple`, `height`

## Demo Page Variations

Six configurations at `/design-system/flare/wobble-rings`:
- Default dark navy (interactive)
- Forest green with tight 16px gap
- Light background with dark rings
- Centered purple origin with wide 35px gap
- Expressive gold with strong wobble and hover
- Banner VC brand colors with lower-right origin
