

To the extent possible, we like to default to HTML & CSS, and plain JavaScript. 

We avoid React, React Patterns, and JSX or TSX syntax.

Astro -- for Static Site Generation.
Tailwind -- for rapid improvised styles. 
Svelte -- for more advanced interactivity and server side rendering.
GSAP -- for any advanced animations, though we try to avoid using GASP and instead use CSS animations and transitions where possible.

Prefer Markdown files for content and data, JSON used sparingly.
